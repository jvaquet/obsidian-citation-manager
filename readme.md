# Obsidian Citation Manager
Implements basic citation management functionality for Obsidian, taylored to my preferences.

The code is not production-ready, use at your own risk.

## Functionality from other projects
Core functionalities are taken from other great obsidian plugins, including:
- [Biblib](https://github.com/callumalpass/obsidian-biblib): Zotero Connector Browser plugin integration
- [Obsidian Zotero Integration](https://github.com/obsidian-community/obsidian-zotero-integration/)/ [pdf2annots](https://github.com/0xabu/pdfannots): Import annotations from PDF
- [Obsidian Smart Link Alias](https://github.com/vpcano/obsidian-smart-link-alias): For nice looking links to literature notes

Make sure to check them out!
